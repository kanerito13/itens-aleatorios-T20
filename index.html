Como usar o site:

  1. Utilize a tabela de tesouro por nivel de desafio do livro base normalmente, se quiser apenas sortear um item por outro motivo pule essa etapa.

  2. Selecione a categoria do item usando o botão "CATEGORIA DE ITEM" abaixo (se tirar "acessório" como categoria e estiver rolando tesouro não mágico use o botão novamente).

  3. Se tirar "item geral" como categoria, use o botão "CATEGORIA DE ITEM GERAL" para definir o seu tipo.

  4. Após definir a categoria, utilize os botões das sessões abaixo para sortear um item (e seus aprimoramentos e encantos se houver).

  5. Se um aprimoramento ou encanto de um item requer um pré-requisito ou outra especificação, sorteie novamente.

  6. É possivel tirar o resultado "material especial" ao sortear um item aprimorado, se acontecer use o botão "MATERIAL ESPECIAL".

  7. É possivel tirar o resultado "item específico" ao sortear um item encantado, se acontecer use a sessão de itens específicos.

  8. As categorias de itens alquímicos (preparados, catalisadores e venenos) estão todas no mesmo botão, se quiser uma categoria específica sorteie um item até receber o tipo procurado.

  9. Os itens de livros que não são o livro base (ameaças, deuses e heróis de arton) possuem uma indicação de qual livro pertencem.

  10. Se o item sorteado não fizer sentido com o ambiente da sua aventura (como um prato especial numa caverna ou um equipamento de aventura que só existe em uma região específica), pode sortea-lo novamente se quiser.
